Linear Search Algorithm (C++)




Linear Search is the most basic searching algorithm out there. To find our desired element we just start from any one of the ends of the array, and iterate through it till we find it.

There is nothing much more too it. Just start from one end, let's say the starting Index 0, then keep iterating through the all elements from the start of the array till we find our desired element. And when we do find the required element, we just stop right there at that iternation and return the index as the position of that element in that array. But if the required element is not in the array, then we just simply return -1
