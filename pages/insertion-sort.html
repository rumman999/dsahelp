Insertion Sort Algorithm (C++)

Insertion Sort Algorithm is another simple algorithm that takes an element of an unsorted array and puts it in the correct position of the sorted part of the array. It is usually compared with sorting playing cards in your hands. There are two group of cards: one sorted and another unsorted. We pick a card from the unsorted part and put it in correct position of the sorted part of cards.

We assume the first element of the array to be sorted. And then we take the second element and compare it with the first element, if the first element is greater, then we swap them. Then we pick the third element of the unsorted array and compare it with the first two elements of the sorted part of the array and put it in its correct position. We repeat this until the whole array is sorted.
